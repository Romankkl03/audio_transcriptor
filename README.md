Сервис по транскрибации аудио. 

Возможности пользователя:

* Пользователь имеет возможность создать свой аккаунт или войти по email и паролю.
* Пользователь может узнать и пополнить баланс своих кредитов.
* Пользователь загружает или записывает (пока не уверен насчет этого) аудиофайл, модель сама определяет язык и делает транскрибацию.
* Есть возможность выбрать язык записи для ускорения процесса перед получением результата.
