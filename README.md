Сервис по транскрибации аудиофайлов. 

Возможности пользователя:

* Пользователь имеет возможность создать свой аккаунт или войти по email и паролю.
* Пользователь может узнать и пополнить баланс своих кредитов.
* Пользователь загружает аудиофайл (.mp3), модель сама определяет язык и делает транскрибацию.
* Есть возможность получить историю предсказаний модели для каждого пользователя.

В качестве модели взял whisper tiny на 39M параметров. 

Для того, чтобы все собралось, необходимо включить ВПН. Это нужно сделать только на стадии docker compose build, чтобы модель загрузилась. 
Затем отключаем впн и поднимаем контейнеры.

<b>Запуск приложения</b>

- docker compose build

- docker compose up -d --scale worker=2
